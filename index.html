Chi Sono

Ciao, sono Gabriel Lazzara, full-stack developer con focus su siti professionali, web app interattive e progetti consegnati a clienti reali.

Lavoro con JavaScript · React · Python · PHP e stack moderni (Tailwind, API REST, deploy su VPS e GitHub Pages). Ogni progetto parte da obiettivi chiari: visibilità online, conversioni e esperienza utente fluida su mobile.

Dal prototipo al go-live: struttura SEO, performance, form di contatto e manutenzione. Esempio recente: sito completo per Rent Boat Mondello (noleggio barche, escursioni, prenotazioni).
